Example 2 : Consider a pipe that sums given array of fibonacci sequence

Checkout source of this example

If the input object is mutable (like here we are pushing new values in the same array object), we must use pipe pure = false

Try Compared to

applyPure Pipe [ works with but not ]

Even though sum() is a pure function, angular puts another restriction on pure pipes; the input must change by reference (new array object)

SUM of {{fib | json}} = {{fib | applyPure: sum}}

apply Pipe [ works with both and ]

Since apply pipe is pure = false, with standard change detection, it works with both, changes to existing fib array or with new array object

SUM of {{fib | json}} = {{fib | apply: sum}}
